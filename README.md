# 전시장 안내 시스템 v14 - 엑셀 자동 배치

## 실행

```bash
python -m http.server 8000
```

- 사용자: http://localhost:8000/
- 관리자: http://localhost:8000/admin.html

## 엑셀 자동 인식 규칙

- 한 셀 = 1m × 1m
- 병합된 유색 셀 + 셀 값 = 부스
- 빈 흰색 셀 = 이동 가능한 복도
- 상하좌우 이동만 허용
- 부스와 복도가 맞닿은 모든 지점을 출입 후보로 생성
- 현재 위치에서 가장 가까운 출입 후보를 자동 선택
- 같은 부스번호가 여러 병합영역에 있으면 하나의 목적지로 통합

## 사용 순서

1. 관리자 화면에서 `부스배치도 XLSX 파일` 선택
2. 자동 분석 결과 확인
3. 사용자 화면 새로고침
4. `지도에서 현재 위치 선택`으로 흰색 복도 지정
5. 부스번호 검색 후 길찾기
