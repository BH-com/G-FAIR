자동 저장 준비
v9: 경로·부스·지점 편집을 모두 같은 MapLibre 지도 방식으로 통일했습니다.

경로 편집

선분을 클릭해 폐쇄·재개하고 사용자 길찾기에 즉시 반영합니다.

부스 편집

부스 높이·색상·특별부스 유형과 기업 정보를 관리합니다.

지점 편집

QR·입구·출구·무대 위치를 드래그하고 추가·삭제합니다.

설정/데이터

전시회 기본설정, 데이터 불러오기와 프로젝트 파일을 관리합니다.

콘텐츠 관리

제품 설명, 무대 프로그램, PDF 링크를 관리합니다.

경로 편집

부스 배치가 보이는 MapLibre 2D 지도 위에서 기존 경량 벡터 편집기를 사용합니다. 빈 공간은 지도 이동, 선분·꼭지점은 경로 편집입니다.

전체 경로0
폐쇄 경로0
경로나 꼭지점을 선택하세요.

부스 편집

부스를 클릭해 선택합니다. 여러 부스를 합칠 때는 Ctrl 키를 누른 채 클릭해 복수 선택하세요. 같은 부스번호로 떨어져 있는 도형은 하나를 선택한 뒤 새 번호를 입력해 분리할 수 있습니다.

부스 도형을 선택하세요.

지점 편집

경로를 보면서 지점을 추가하거나 이동합니다. 자석 기능을 켜면 가장 가까운 경로 위로 자동 배치됩니다.

지점을 선택하세요.

기본설정

사용자 화면과 관리자 상단에 표시할 전시회명을 설정합니다.

지도 기본설정

사용자 지도에 표시되는 모든 부스의 3D 높이를 한 번에 설정합니다. 부스 편집 지도는 2D로 유지됩니다. 현재 전시장 좌표 축척에서는 80~150 정도가 일반적인 3D 높이입니다.

프로젝트 관리

project.json 저장·불러오기와 GitHub 배포용 ZIP 생성을 관리합니다.

부스 배치도

기존 셀 색상·테두리·병합 셀 방식의 부스 배치도 XLSX를 분석하여 부스, 통로, 기본 경로를 만듭니다.

부스 배치도 양식
파일을 선택하면 먼저 분석 결과를 확인한 뒤 적용할 수 있습니다.

기업 데이터

부스번호, 기업명, 품목, 제품 설명, 홈페이지 열을 자동 인식합니다.

프로그램 일정

무대, 날짜, 시작, 종료, 프로그램명, 설명 열을 불러옵니다.

기업 제품 설명

프로젝트 저장
